# RAG Benchmark

This folder contains code and benchmarks for evaluating RAG (Retrieval-Augmented Generation) approaches.

## Contents
- **RAG benchmark.ipynb**: Core notebook containing the RAG benchmarks.
- **src/**: Source code for embeddings, retrieval, dataloading, vectorized storing, etc.
- **data/**: Sample datasets (	est.jsonl, 	rain.jsonl, alid.jsonl) used for RAG evaluation.
