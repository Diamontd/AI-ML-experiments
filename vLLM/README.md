# vLLM Agent

This folder contains the vLLM agent notebook and related validation datasets for evaluating the agent using exact-match and LLM-as-a-Judge approaches.

## Contents
- **vLLM agent.ipynb**: Core notebook implementing and evaluating the vLLM agent.
- **data/**: Contains validation datasets used by the notebook and scripts.
- **scripts/**: Audit and coverage extractor scripts for the dataset.
